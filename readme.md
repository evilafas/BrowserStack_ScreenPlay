# BrowserStack ScreenPlay

Proyecto de automatización de pruebas usando Serenity BDD y Screenplay.

## Descripción

Este proyecto contiene pruebas automatizadas para validar algunos flujos de la aplicación usando BrowserStack.

## Tecnologías

- Java
- Serenity BDD
- Cucumber
- Screenplay
- BrowserStack

## Requisitos

- Tener Java instalado
- Tener Gradle instalado
- Tener acceso a BrowserStack

## Ejecución

Para ejecutar las pruebas se debe correr el proyecto desde el IDE o por consola.

```bash
gradle test
